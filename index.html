1. Token Volume Calculus

Calculates the total tokens processed per case based on base agent usage and synthesis overhead.

Total Input Tokens = (Base Tokens * 0.85) + Synthesis Input
Total Output Tokens = (Base Tokens * 0.15) + Synthesis Output

2. On-Demand Economics (Variable Cost)

Calculates the fully loaded pay-as-you-go cost including infrastructure overhead.

Input Cost = (Total Input / 1M) * Provider Input Rate * Monthly Volume
Output Cost = (Total Output / 1M) * Provider Output Rate * Monthly Volume
Base Model Cost = Input Cost + Output Cost
Infra Overhead = Base Model Cost * Infra Burden %

Total Monthly Cost = Base Model Cost + Infra Overhead

3. Hosted Economics (Fixed Cost)

Calculates the dedicated server flat-rate costs assuming 100% uptime.

Compute Cost = Hardware Rate/min * 43,200 min/mo

Total Fixed Cost = Compute Cost + Storage Cost

4. Break-Even Intersection

Calculates the exact volume threshold where building becomes cheaper than buying.

Cost Per Case = Total On-Demand Monthly Cost / Monthly Volume

Break-Even Volume = Total Hosted Fixed Cost / Cost Per Case
